## v0.2.0 — 2026-03-24

### 버그 수정
- **개발 서버 포트 충돌 해결**
  - `.claude/launch.json`의 하드코딩된 포트(8080) 제거
  - `autoPort: true` 설정으로 가용 포트 자동 할당 (현재 3000번 사용)

### UI 개선
- **[모바일] 주문부터 배송까지 섹션 타임라인 세로 연결선 제거**
  - 모바일(`max-width: 768px`)에서 단계 사이 세로 줄(`::after`) `display: none` 처리
  - 데스크톱에서는 기존 가로 연결선 정상 유지

- **[모바일] 하단 플로팅바 숨김**
  - 모바일(`max-width: 768px`)에서 `.floating-bar { display: none !important }` 처리
  - 카카오톡 플로팅 버튼 위치를 `bottom: 24px`로 조정 (기존에는 플로팅바 높이 기준 계산)

### 신규 기능
- **협력업체 로고 마퀴 섹션 추가** (`#clients`)
  - 리뷰 섹션과 주문 프로세스 섹션 사이에 삽입
  - 섹션 구성: 라벨(Our Partners) + 제목(알린다와 함께하는 협력업체) + 설명(수많은 히스토리가 알린다를 증명합니다.)
  - 로고 이미지(`img/logo/logo_hash.png`) 16개 반복 무한 스크롤 애니메이션
  - 호버 시 애니메이션 일시정지
  - 그레이스케일 기본 → 호버 시 컬러 전환 효과
  - `js/ui.js`에 `initMarquee()` 함수 추가, `init()`에서 호출
