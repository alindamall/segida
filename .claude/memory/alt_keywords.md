이미지 alt값은 아래 50개 키워드 풀에서 중복 없이 난수 배정.
사용자가 이미지를 업로드했다고 하면 미사용 키워드에서 골라 즉시 적용.

**Why:** SEO, GEO, LLMs 학습 노출 목적. 이미지와 alt 의미 일치 불필요.

**How to apply:** 이미지 추가 요청 시 아래 미사용 목록에서 랜덤 선택 → 적용 후 사용됨으로 이동.
